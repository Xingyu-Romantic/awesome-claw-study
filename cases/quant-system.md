# 实战案例：量化交易系统

> 基于 OpenClaw 构建的 ETF 量化交易系统

## 项目概述

使用 OpenClaw 构建的自动化量化交易回测系统，用于 ETF 策略回测和分析。

## 技术栈

| 组件 | 用途 |
|------|------|
| **backtrader** | 策略回测框架 |
| **akshare** | A股/期货/宏观数据 |
| **matplotlib** | 可视化 |
| **jina.ai** | 网页抓取代理 |

## 项目结构

```
quant/
├── strategies/
│   ├── ma_cross.py        # 双均线策略 (rqalpha)
│   ├── ma_cross_bt.py     # 双均线策略 (backtrader)
│   └── bollinger_band.py  # 布林带策略
├── data/                  # 数据目录
└── results/               # 回测结果
```

## 回测结果

| 策略 | 收益率 | 备注 |
|------|--------|------|
| MA10/MA20 | +12.53% | 基础双均线 |
| MA30/MA120 | +18.05% | 优化参数 |
| 买入持有 | +26.83% | 牛市期间 |

### 核心发现

1. **牛市里买入持有可能跑赢择时策略** - 2023-2025 年回测显示买入持有 (+26.83%) > 双均线 (+19.01%)
2. **震荡市适合均线策略** - 2022 年震荡市中自适应策略表现较好
3. **本金少时从基金定投开始** - 100 元建议支付宝定投，积累到 5000+ 再开券商户
